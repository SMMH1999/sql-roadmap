# SQL Roadmap for Data Analytics

This repository tracks my progress while learning SQL from zero to a professional level, following the SQL roadmap from [roadmap.sh](https://roadmap.sh/sql).

## Tech Stack

- Database: PostgreSQL
- IDE: JetBrains DataGrip
- OS: Windows
- Version Control: Git + GitHub Desktop

## Folder Structure

- `00-setup/` – Installation and environment setup
- `01-basics/` – SQL basics (SELECT, WHERE, ORDER BY, LIMIT, etc.)
- `02-intermediate/` – JOINs, GROUP BY, subqueries
- `03-advanced/` – Window functions, CTEs, advanced aggregations
- `04-expert/` – Performance tuning, indexing, execution plans
- `05-project/` – Final retail analytics case study

Each folder contains:
- `notes.md` – Concept summaries
- `exercises.md` – Practice questions
- `solutions.sql` – My solutions
